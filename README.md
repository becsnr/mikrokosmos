### 🌌 Mikrokosmo Sweets
Uma doceria cósmica interativa, onde cada doce parece ter vindo direto de outra galáxia.

---

### ✨ Sobre o projeto
O Mikrokosmo Sweets é uma aplicação web que simula o site de uma doceria com temática cósmica.
A proposta é unir design criativo com desenvolvimento front-end, criando uma experiência visual imersiva e agradável.

---

### 🍦 Funcionalidades
- Navegação entre seções (Home, Cardápio, Sobre e Contato)
- Cardápio com categorias (destaques, sorvetes, bolos, bebidas, etc)
- Cards de produtos com imagem, descrição e preço
- Interface responsiva (desktop e mobile 📱)
- Design dinâmico com identidade visual temática (galáxia)

---

### 🎨 Interface
O design foi inspirado no universo espacial, com cores suaves em tons de lilás e elementos como planetas, órbitas e estrelas.

---

### 🛠️ Tecnologias utilizadas
- React
- JavaScript
- CSS

---

### 📱 Responsividade
O projeto foi desenvolvido para funcionar bem tanto em desktop quanto em dispositivos móveis, mantendo a experiência visual consistente.

---

### 🚀 Como rodar o projeto
```bash
git clone https://github.com/becsnr/mikrokosmos.git
cd mikrokosmo
npm install
npm run dev
```
